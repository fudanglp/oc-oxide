oc-oxide Architecture

Rust/Tauri OpenConnect helper with a daemon-owned VPN control plane

Trust Boundary

  • - Desktop and CLI stay unprivileged.
  • - Daemon gates IPC with peer credentials and polkit.
  • - Secrets are transient except opt-in keyring passwords.
  • - Runtime journal records reversible non-secret state.

Tunnel Lifecycle

  • - One tunnel worker owns all libopenconnect calls.
  • - Auth prompts flow back to clients as typed IPC events.
  • - Policy planner applies TUN, routes, then DNS.
  • - Disconnect reverts DNS, routes, then TUN.

Persistent Data

  • - Profiles are TOML and must remain non-secret.
  • - GitHub Sync stores private repo snapshots only.
  • - Access tokens stay in memory; refresh tokens use keyring.
  • - Diagnostics redact auth and profile secret material.
